01 — CONCEPT

What is Simple Linear Regression?

Simple Linear Regression (SLR) is a supervised machine learning algorithm that models the linear relationship between one independent variable X (input) and one continuous dependent variable Y (output). It fits a straight line through the data points to best predict future values.

📐
DEFINITION
A statistical method that finds the best-fit line through data points, expressing Y as a linear function of X: ŷ = β₀ + β₁x. The goal is to minimize prediction error.
🎯
PURPOSE
Predict unknown values of a dependent variable, understand the strength and direction of a relationship between two variables, and make data-driven decisions.
🌍
REAL-WORLD USE
House price prediction (area → price), salary estimation (experience → salary), weather forecasting, medical dosage estimation, stock trend analysis.
⚠️
ASSUMPTIONS
Linearity between X and Y, independence of observations, homoscedasticity (constant variance), and normally distributed residuals (errors).

Key Parameters & Variables

Every parameter plays a crucial role in building the regression model.

β₀ — Intercept
β₀ = ȳ - β₁x̄
Value of Y when X = 0. It's the point where the regression line crosses the Y-axis.
β₁ — Slope
β₁ = Σ(xᵢ-x̄)(yᵢ-ȳ) / Σ(xᵢ-x̄)²
Rate of change of Y per unit change in X. Positive slope = positive relationship.
SSE — Sum of Squared Errors
SSE = Σ(yᵢ - ŷᵢ)²
Total squared difference between actual and predicted values. Lower is better.
R² — Coefficient of Determination
R² = 1 - SSE/SST
How well the model explains variance in Y. R² = 1 is a perfect fit; R² = 0 means no fit.
MSE — Mean Squared Error
MSE = SSE / n
Average squared prediction error across all data points. Key metric for model evaluation.
Pearson r — Correlation
r = Σ(x-x̄)(y-ȳ) / √[Σ(x-x̄)²·Σ(y-ȳ)²]
Measures linear relationship strength: −1 (perfect negative) to +1 (perfect positive).

Step-by-Step Explanation

#STEPDESCRIPTION
1Collect DataGather paired (X, Y) data points — your dataset.
2Compute MeansCalculate x̄ (mean of X) and ȳ (mean of Y).
3Compute DeviationsFind (xᵢ − x̄) and (yᵢ − ȳ) for each data point.
4Compute β₁β₁ = Σ(xᵢ−x̄)(yᵢ−ȳ) ÷ Σ(xᵢ−x̄)² — the slope.
5Compute β₀β₀ = ȳ − β₁·x̄ — the intercept.
6Build the ModelWrite the equation: ŷ = β₀ + β₁·x
7PredictSubstitute new X values to get predicted Y (ŷ).
8EvaluateCompute SSE, MSE, and R² to assess model quality.

Worked Example with Small Dataset

Given: Study hours (X) and Exam scores (Y) for 5 students.

TOTALS8510
  1. Compute means: x̄ = (1+2+3+4+5)/5 = 3  |  ȳ = (55+60+70+75+90)/5 = 70
  2. Compute slope β₁: β₁ = Σ(xᵢ−x̄)(yᵢ−ȳ) / Σ(xᵢ−x̄)² = 85 / 10 = 8.5
  3. Compute intercept β₀: β₀ = ȳ − β₁·x̄ = 70 − 8.5 × 3 = 70 − 25.5 = 44.5
  4. Final regression equation: ŷ = 44.5 + 8.5x
  5. Predict score for 6 hours of study: ŷ = 44.5 + 8.5 × 6 = 44.5 + 51 = 95.5
